# LLM Project Declaration

This repository is an **LLM project**.

Initial project scaffolding, implementation, and documentation were generated with LLM assistance and are intended to be reviewed, tested, and maintained by a human owner.

## Scope of generated implementation

- Python CLI for GitHub authentication and repository inventory export
- YAML-based desired-state sync (`present`, `archived`, `absent`)
- Basic documentation and TODO backlog

## Human review checklist

- Verify token scopes and security practices
- Validate dry-run and apply behavior in a safe account first
- Add tests before production use
- Protect important repositories before using `absent`
